## 🇪🇸 Español

### 📌 Descripción

Portafolio profesional personal, construido desde cero con HTML, CSS y JavaScript puro (sin frameworks). Presenta mi trayectoria como Ingeniera de Sistemas, mis proyectos, habilidades técnicas y formas de contacto, con un diseño responsivo, modo oscuro y soporte bilingüe (ES/EN).

🔗 **Demo en vivo:** [carmenmirna.vercel.app](https://carmenmirna.vercel.app/)

### ✨ Características

- 🌗 **Modo oscuro/claro** con persistencia (recuerda tu preferencia usando `localStorage`)
- 🌍 **Cambio de idioma** Español/Inglés sin recargar la página
- 🎬 **Animaciones al hacer scroll** (fade-in progresivo en tarjetas y secciones)
- 🖼️ **Galería interactiva** de capturas por proyecto, con lightbox (vista ampliada)
- 📊 **Estadísticas de GitHub** en tiempo real (commits, lenguajes más usados)
- ⏳ **Loader de carga** elegante que espera a que todo el contenido esté listo
- 📱 **100% responsivo**, adaptado a celulares, tablets y escritorio
- ♿ Considera accesibilidad (`prefers-reduced-motion`, buen contraste, etiquetas `aria-label`)

### 🛠️ Stack Tecnológico

| Frontend | Herramientas |
|---|---|
| HTML5 semántico | Git & GitHub |
| CSS3 (variables, grid, flexbox) | Vercel (hosting/despliegue) |
| JavaScript (vanilla, sin frameworks) | Font Awesome (iconos) |
| Google Fonts (DM Serif Display, Inter, JetBrains Mono) | GitHub Stats API (widgets externos) |

## 🇺🇸 English

### 📌 Description

Personal professional portfolio, built from scratch with plain HTML, CSS, and JavaScript (no frameworks). It showcases my journey as a Systems Engineer, my projects, technical skills, and contact channels, with a responsive design, dark mode, and bilingual support (ES/EN).

🔗 **Live demo:** [carmenmirna.vercel.app](https://carmenmirna.vercel.app/)

### ✨ Features

- 🌗 **Dark/Light mode** with persistence (remembers your preference via `localStorage`)
- 🌍 **Language switcher** Spanish/English with no page reload
- 🎬 **Scroll-triggered animations** (progressive fade-in on cards and sections)
- 🖼️ **Interactive project gallery** with lightbox (full-size view)
- 📊 **Live GitHub stats** (commits, most-used languages)
- ⏳ **Elegant page loader** that waits until all content is fully loaded
- 📱 **Fully responsive**, adapted for mobile, tablet, and desktop
- ♿ Accessibility-conscious (`prefers-reduced-motion`, solid contrast, `aria-label` tags)

### 🛠️ Tech Stack

| Frontend | Tools |
|---|---|
| Semantic HTML5 | Git & GitHub |
| CSS3 (custom properties, grid, flexbox) | Vercel (hosting/deployment) |
| Vanilla JavaScript (no frameworks) | Font Awesome (icons) |
| Google Fonts (DM Serif Display, Inter, JetBrains Mono) | GitHub Stats API (external widgets) |

### 🚀 Running it locally

```bash
# Clone the repository
git clone https://github.com/carmenmirna-is/Portafolio.git

# Enter the folder
cd Portafolio

# Open it with Live Server (VS Code) or any local server
# No dependencies to install: it's plain HTML/CSS/JS
```
